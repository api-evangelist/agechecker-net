# AgeChecker.Net (agechecker-net)
Our API can be used to provide a seamless verification experience for most customers. Send the customers data directly to us and only show the popup if a photo ID is required. Most customers wont even know they were verified, and those who require photo ID verification can get verified easily with our simple upload options.
